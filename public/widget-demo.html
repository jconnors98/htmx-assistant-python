METHOD 3 RECOMMENDED

JavaScript Widget Loader

The easiest way to embed - just add one script tag:

<!-- Add this anywhere in your HTML, preferably before </body> -->
<script 
  src="https://bcca.ai/flask/widget-loader.js" 
  data-mode="YOUR_MODE_ID" 
  data-theme="#82002d"
  data-position="bottom-right"
  data-base-url="https://bcca.ai/flask"
></script>

Configuration Options:

Attribute Required Description Default
data-mode Yes Your assistant mode ID -
data-theme No Primary color (hex format) #82002d
data-position No Widget position on screen bottom-right
data-base-url No API base URL (for custom domains) Current origin

Position Options:

  • bottom-right (default)
  • bottom-left
  • top-right
  • top-left
